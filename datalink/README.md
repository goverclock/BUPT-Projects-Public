本科二年级,<<计算机网络>>课程实验. 

利用所学数据链路层原理,自己设计一个滑动窗口协议,在仿真环境下编程实现有噪音信道环境下两站点之间无差错双工通信.

信道模型为8000bps全双工卫星信道,信道传播时延270毫秒,信道误码率为1e-5,信道提供字节流传输服务,网络层分组长度固定为256字节.
